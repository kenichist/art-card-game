Matching System

Match items with collectors to earn values based on matching descriptions.

How It Works

  1. Each person creates two collector items with unique descriptions and values.
  2. The auctioneer presents one item at a time to all collectors.
  3. If a collector item matches descriptions with the presented item, each matching description adds to the money given.
  4. For example: If an item matches "Like postmodernism products" (21w) and "Like transportation" (13w), the collector gets 21+13=34w.
  5. The items and collector cards are matched by ID to ensure proper tracking.
